# NeoFMS

> Learn the system. Not the buttons.

NeoFMS is an open-source Flight Management System (FMS) trainer focused on the Airbus A320neo.

The goal is not simply to reproduce the MCDU interface, but to provide a realistic learning environment that helps pilots understand how an Airbus Flight Management System works.

---

## Philosophy

NeoFMS is built around three core principles:

- Fidelity over appearance.
- Quality over speed.
- Understanding over memorization.

---

## Features (Planned)

- Airbus A320neo MCDU
- Interactive Academy
- Check Rides
- Dynamic Scenarios
- SimBrief Integration
- Progressive Learning System
- Offline PWA
- Accessibility Support
- Multiple Languages
- Theme Engine
