# NeoFMS Specification

## Goal

NeoFMS aims to reproduce the operational behaviour of an Airbus A320neo MCDU while remaining an educational platform.

---

## Display

- 24 columns
- 14 lines
- Scratchpad on the last line
- Independent Theme Engine

---

## Keyboard

- Full Airbus layout
- Functional keys
- Alphanumeric keys
- CLR behaviour identical to the aircraft

---

## Modes

- Free Mode
- Academy
- Check Ride
- Scenario Mode

---

## Supported Aircraft

Initially:

- Airbus A320neo

Future:

- A319
- A321
- A330
